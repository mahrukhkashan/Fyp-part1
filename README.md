# Explainable Multimodal AI-based Sepsis Prediction and Assistance System

A comprehensive system for early sepsis prediction with explainable AI and conversational assistance.

## Features

- **Early Sepsis Prediction**: Machine learning models to predict sepsis risk
- **Multimodal Data Integration**: Combines structured (vitals, labs) and unstructured (clinical notes) data
- **Explainable AI**: SHAP-based explanations for model predictions
- **Conversational Assistant**: AI chatbot for patient education and support
- **Dual Interface**: Separate dashboards for clinicians and patients
- **Real-time Monitoring**: Continuous patient monitoring and alerts

## Prerequisites

- Python 3.8+
- PostgreSQL 13+
- MIMIC-III database (optional, sample data included)
- 8GB+ RAM recommended
